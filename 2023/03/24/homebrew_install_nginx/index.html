mac系统通过homebrew安装nginx

2023/03/24 编程人生 nginx 共 618 字，约 2 分钟

准备

  • mac系统
  • homebrew 环境已经配置好

操作步骤

  1. 查看本地是否安装nginx
     brew search nginx

    nginx上面没有显示✓,表示nginx没有安装

  2. 查看nginx信息
     brew info nginx

    如图所示,显示了nginx的版本. 也明确说明了系统没有安装nginx

  3. 安装nginx
     brew install nginx

    安装nginx成功.
    nginx默认监听8080端口, 配置文件路径也给出来了 /opt/homebrew/etc/nginx/nginx.conf
    同时也给了启动nginx的命令

    brew services start nginx

4.验证安装成功 此时,nginx命令已经生效了.

 nginx -v

当然, 也可以执行下面命令查看更详细的信息

nginx -V

至此,nginx已经安装成功了. 可以通过去配置文件修改相应配置, 启动nginx了

启动nginx

# 启动
nginx
brew services start nginx

启动成功了,此时可以通过 http://localhost:8080 验证一下

补充nginx的常用命令

# 查看版本
nginx -v
nginx -V

# 启动
nginx # 加载默认配置
nginx -c 配置文件路径 # 加载指定配置文件
brew services start nginx


# 停止
nginx -s stop # 快速停止
nginx -s quit # 安全停止

# 不重启加载配置
nginx -s reload 

# 验证配置
nginx -t

好的, 接下来尽情使用自己的nginx服务器吧!
